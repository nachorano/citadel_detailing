Limpiadores

Inferno Gel

600 ml

APC Premium desarrollado para los más exigentes detailers del mercado. Posee un sistema de suspensión de suciedad y gran poder espumógeno con agradable fragancia a rosas.

Modo de Uso
  • Diluir el Producto según Requerimiento de Uso.
  • Rociar la Superficie o el Elemento de Trabajo.
  • Trabajar la Superficie agitando el Producto.
  • Recomendamos usar Cepillo.
  • Retirar el Producto con una Microfibra Seca o Húmeda

  • Rociar la Superficie o el Elemento de Trabajo.
  • Trabajar la Superficie agitando el Producto.
  • Recomendamos usar Cepillo.
  • Retirar el Producto con una Microfibra Seca o Húmeda

Formule Conquest

  • Rociar la Superficie o el Elemento de Trabajo.
  • Trabajar la Superficie agitando el Producto.

X Tar

600 ml

Removedor de brea con excelente poder de limpieza. Es seguro para utilizarse en llantas, pintura, vidrios y cromados. Contenido listo para usarse.

Modo de Uso
  • Rociar sobre la Superficie o el Elemento a trabajar.
  • Dejar actuar por 1 Minuto aproximadamente.
  • Retirar el Producto con una Microfibra Húmeda ejerciendo una Leve Presión.
  • Revisar y de ser necesario Repetir el Proceso.

Bug Remover

600 ml

Limpiador exclusivo para remover insectos con gran precisión, fácil de usar y seguro para ser aplicado en todo tipo de superficies.

Modo de Uso
  • Rociar sobre la Superficie o el Elemento a trabajar en Frío.
  • Dejar actuar por unos Minutos.
  • Retirar el Producto con Agua a Presión o Hidrolavadora.
  • Revisar y de ser necesario Repetir el Proceso.
  • En caso de ser necesario, utilizar Esponja Quita Insectos o Cepillo de Cerdas Suaves.

Clean Vision

600 ml

Limpiador con agentes anti-fog que previene el empañado de vidrios. Posee propiedades desengrasantes que brindan una máxima remoción sin esfuerzo. Fórmula con fragancia a limón.

Modo de Uso
  • Agitar el Producto.
  • Rociar sobre la Microfibra o Superficie a trabajar.
  • Utilizar la Microfibra 1 para aplicarlo. (1 Vidrio a la vez)
  • Utilizar la Microfibra 2 para retirar el excedente.
  • Realizar la aplicación un Vidrio a la vez hasta completar el Vehículo.
  • NO aplicar en Plásticos, Llantas o Pintura.

Surface Desinfect

600 ml

Desinfectante tensioactivo, bactericida e inhibidor de actividad viral. Posee alcohol, no mancha ni genera daños en la piel. Aditivos agregados para ser amigable con múltiples superficies.

Modo de Uso
  • Agitar el Producto.
  • Rociar sobre la Superficie a Trabajar.
  • Utilizar la Microfibra para aplicarlo hasta evaporar el Producto.

Iron Warning

600 ml

Descontaminante férrico de llantas el cual actúa eficazmente sobre el polvo de las pastillas de freno. Posee un gran poder de limpieza. Fórmula no diluible, listo para usar.

Modo de Uso
  • Usar guantes para no tener contacto con el Producto.
  • Rociar sobre la Superficie o Elemento de Trabajo. (Siempre Frío)
  • Trabajar la Superficie agitando el Producto. (Recomendamos usar Cepillo)
  • Retirar el Producto con Agua. (Sea Presión o Manguera)
